# Get Started (No Technical Experience Needed)

Workflow Link Organizer sets itself up for you. You don't need to know Python,
the command line, or anything technical — **Claude Code does the work.** You just
follow four short steps and then tell it to set things up in plain English.

---

## What you need (one-time, ~10 minutes)

### 1. Install Python
- Go to **https://www.python.org/downloads/**
- Click the big yellow **Download Python** button, then run the installer.
- **Important:** on the first screen of the installer, check the box that says
  **"Add python.exe to PATH"** before clicking Install. (If you miss it, that's
  okay — Claude Code can help fix it.)

### 2. Install Claude Code
- Go to **https://claude.com/claude-code** and follow the install instructions
  for Windows.
- This is the tool that will set everything up for you.

### 3. Download this project
- On the project's GitHub page, click the green **Code** button, then
  **Download ZIP**.
- Find the downloaded ZIP in your Downloads folder, right-click it, and choose
  **Extract All**. Pick a folder you'll remember, like your Documents folder.

### 4. Open the folder in Claude Code
- Open Claude Code.
- Point it at the folder you just extracted (the one containing this file).

---

## Now just ask

In Claude Code, type this and press Enter:

> **Set up Workflow Link Organizer for me.**

Claude Code will read the included instructions, check that everything it needs
is installed, install the app, and start it. If anything is missing or goes
wrong, it will tell you what to do in plain language. When it's done, the app
opens in your web browser automatically.

That's it. 🎉

---

## After it's running

- The app lives at **http://127.0.0.1:5757** in your browser. Bookmark that page.
- To open it again later, double-click **Start.bat** in the project folder.
- To get the most out of it, set up separate browser profiles (one per client or
  project). The **README.md** file explains how, and Claude Code can walk you
  through it — just ask: *"Help me set up browser profiles."*

## If you get stuck

Ask Claude Code. Some things you can say:
- *"The setup gave an error — can you fix it?"*
- *"It says Python isn't installed."*
- *"How do I start the app again?"*
- *"Walk me through creating a Chrome profile for a new client."*
